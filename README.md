European B2B Sales Analytics Dashboard

Tools: Power BI Desktop, DAX, Power Query
Data: Synthetic dataset — 500 orders, 10 European countries, 2022–2024

About the Project
This dashboard analyzes B2B sales performance of a security systems distributor across European markets.

What was done
Designed a Star Schema: Fact_Sales + Dim_Product + Dim_Distributor
Cleaned and transformed data in Power Query
Built DAX measures: Total Revenue, Units Sold, Profit Margin
Created interactive visualizations with Country slicer

Dashboard includes
KPI cards — Total Revenue ($1.13M), Total Orders (500), Units Sold (13K), Profit Margin
Revenue by Country — horizontal bar chart
Monthly Revenue Trend — line chart
Revenue by Category — pie chart
Interactive Country filter — slicer

Key Insights
Romania and Italy show highest revenue among all markets
Detector category dominates product sales (32%)
Sales show seasonal patterns with peaks in summer months

Skills demonstrated: Power BI Desktop, Power Query, DAX, Star Schema, data modeling, business analytics
